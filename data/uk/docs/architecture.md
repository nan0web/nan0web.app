# Архітектура

**NaN0Web Engine** — це суверенне середовище виконання, яке перетворює структури даних на інтерактивні інтерфейси.

## Основні концепції

- **Model-as-Schema**: Єдине джерело правди для всіх форм та моделей даних.
- **Intent Delegation**: Раннер може делегувати інтенцій під-додаткам.
- **WebSocket Bridge**: Синхронізація стану в реальному часі між усіма підключеними UI.

## Послідовність

1. `Runner.run()` -> знаходить додатки.
2. `IntentResolver` -> делегує дії користувача.
3. `UI Adapters` -> рендерять вигляд.
