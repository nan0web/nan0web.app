# Документація

Ласкаво просимо до документації NaN0Web Раннера. Цей рушій побудований на принципі **OLMUI Triad**:

1. **Одна Логіка**: Спільний стан та бізнес-правила.
2. **Багато UI**: Різні адаптери (CLI, Web, Chat) рендерять ті самі дані.

## Початок роботи

Щоб запустити свій додаток через раннер:

```bash
nan0web run --dsn data/
```

## Структура даних

Кожен додаток складається з:
- `_ / t.nan0` — Переклади.
- `_ / nav.nan0` — Дерево навігації.
- `index.nan0` — Головна точка входу.
